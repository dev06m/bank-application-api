

## Kurulum ve Çalıştırma

Uygulamayı IDE kullanmadan çalışır hale getirmek için 
aşağıdaki komutları kullanmanız yeterli olacak.

**Windows** 
<br/>cd Desktop/transactions
<br/>mvnw package
<br/>cd target
<br/>java -jar *.jar

**MacOS**
<br/>cd transactions
<br/>./mvnw package
<br/>cd target
<br/>java -jar *.jar
